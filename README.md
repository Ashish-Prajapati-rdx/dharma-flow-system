# 🍃 AyurSutra - Panchakarma Treatment OS

AyurSutra is a production-grade, highly specialized MERN Full-Stack Operating System designed for Ayurvedic clinics and Panchakarma practitioners. It streamlines complex, multi-day clinical operations, tracks patient progression cycles, manages time slots dynamically, and integrates intelligent guardrail-locked AI assistance.

---

## 🚀 Key Features

* **Real-Time Doctor Dashboard:** Tracks live current sessions, active patient graphs, and daily diagnostic metrics without UI state-drop on refresh.
* **Smart Daily Timeline & Scheduling:** Automatically locks and maps patient appointments into precise hourly slots (08:00 AM - 08:00 PM) with built-in layout hydration guards.
* **21-Day Panchakarma Progress Tracker:** Visualizes structural daily health parameters (Digestion, Sleep Quality, Energy Levels) along a dynamic timeline.
* **Background Notification Engine:** Utilizes an asynchronous, non-blocking Nodemailer pipeline to instantly dispatch structured HTML treatment alerts and dietary precautions to patients.
* **Guardrail-Locked AI Chatbot:** Features an integrated Gemini 1.5 Flash assistant customized via strict system instructions to strictly address Ayurvedic protocols, herbs, and precautions while deflecting off-topic queries.

## 🛠️ Tech Stack

### Frontend
* **React.js** (TypeScript)
* **Tailwind CSS** (Custom Emerald/Sage green design system)
* **Axios** (State-managed API consumer requests)

### Backend
* **Node.js** & **Express.js** (TypeScript architecture)
* **Mongoose** (Strict Object Data Modeling)
* **Google Generative AI SDK** (Gemini 1.5 Flash Integration)
* **Nodemailer** (Asynchronous mail transport routing)

### Database
* **MongoDB Atlas** (Cloud-hosted transactional storage)
